修改ip地址

  1. ifconfig 找出网卡的名称 查看网卡为ens33

  2. cd /etc/sysconfig/network-scripts/ —- 查看

  3. vim ifcfg-ens33

重启网络服务

systemctl restart network

vim /etc/NetworkManager/NetworkManager.conf

dns=no
